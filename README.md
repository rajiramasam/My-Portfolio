# My Portfolio Website

## 🚀 About the Project
This is my personal portfolio website showcasing my skills, projects, and experience. The website is built using HTML, CSS, and JavaScript and features an interactive and responsive design.

## 📂 Directory Structure

## 🌟 Features
- **Interactive UI**: Smooth animations and hover effects.
- **Skills Display**: Icons and descriptions from `skills.json`.
- **Responsive Design**: Works across all screen sizes.
- **Sticky Navigation**: Highlights active sections on scroll.
- **Dark & Light Mode**: Adaptive design for better UX.

## 🛠️ Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Libraries & Frameworks**: TailwindCSS, Bootstrap, MaterialUI, Redux
- **Backend & Databases**: NodeJS, ExpressJS, MongoDB, PostgreSQL, MySQL
- **Cloud & Deployment**: AWS, Netlify, DigitalOcean, Heroku
- **Version Control**: Git, GitHub

## 📜 Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/rajiramasam-my-portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd rajiramasam-my-portfolio
   ```
3. Open `index.html` in a browser.
